# 상품 Q&A 템플릿 맵 (2026-04-04)

대표님 실제 답변 패턴 분석을 바탕으로 1차 템플릿을 정리한 문서입니다.
worker는 `src/qna-templates.js` 와 `src/qna-drafts.js` 를 사용합니다.

## 템플릿 코드

- `tax_fee_clear`
  - 용도: 관세/추가지불/관부가세 문의
  - 톤: 짧은 즉답형

- `shipping_fee_variable`
  - 용도: 배송비 변동 문의
  - 톤: 설명형
  - 기준점: 현지 물류비 / 국제배송비 / 환율 / 결제화면

- `shipping_eta_uncertain`
  - 용도: 특정 날짜 도착, 급배송, 통관 변수
  - 톤: 짧은 단정 회피형

- `shipping_eta_general`
  - 용도: 일반 배송기간 문의
  - 톤: 설명형

- `stock_check_needed`
  - 용도: 재고/입고/품절/발송지연
  - 톤: 설명형
  - 패턴: 확인 후 다시 안내

- `option_guidance_general`
  - 용도: 옵션/색상/선택/구매 가능 여부
  - 톤: 설명형

- `components_reference`
  - 용도: 구성품/포함품/세트/단품 문의
  - 톤: 설명형
  - 기준점: 상품 페이지 옵션/상세설명

- `compatibility_no`
  - 용도: 특정 대상만 사용 가능할 때
  - 톤: 짧은 불가 안내형

- `compatibility_check`
  - 용도: 호환 여부 일반 확인
  - 톤: 설명형

- `authenticity_general`
  - 용도: 정품/상품 진위/상품 기준 문의
  - 톤: 설명형

- `size_reference`
  - 용도: 사이즈/실측/국내사이즈 환산 문의
  - 톤: 설명형
  - 패턴: 단정 회피 + 실측 기준 안내

- `size_recommendation_general`
  - 용도: 사이즈 추천 문의
  - 톤: 짧은 안내형

- `usage_check_needed`
  - 용도: 사용법/열기/조립/설치 문의
  - 톤: 설명형
  - 패턴: 확인 후 다시 안내

- `quantity_possible`
  - 용도: 수량 주문 가능 문의
  - 톤: 짧은 가능 안내형

- `followup_talk`
  - 용도: 톡톡 유도/추가 확인 필요
  - 톤: 짧은 전환형

## 운영 원칙

1. 확실한 사실은 짧게 답한다.
2. 일정/사이즈/통관/현지상황은 단정하지 않는다.
3. 상세페이지/실측/결제화면 같은 기준점을 자주 제시한다.
4. 확인이 필요한 재고/현지이슈는 `확인 후 안내` 패턴을 사용한다.
5. 위험 키워드(환불/보상/불량/파손/신고/법적/책임)는 자동 응답 제외로 둔다.
