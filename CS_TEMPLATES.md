# CS_TEMPLATES.md — 고객응대 템플릿 규칙

## 목적
또몽이네 스토어의 고객응대 메시지를 일관된 형식으로 빠르게 작성하기 위한 템플릿 문서다.

## 참조 원본
- 원본 CS 지침은 `references/cs/CS지침-원본.md`, `references/cs/CS지침2-원본.md`에 보관한다.
- 이 문서는 원본 지침 중 실제로 자주 쓰는 고객응대 문구 규칙만 요약한다.

---

## 1. 공통 출력 규칙
- 요청이 모호하면 먼저 `사전안내문 작성`인지 `실제 고객 문의 답변 작성`인지 구분한다.
- 고객 응대 최종 답변은 복사 편의를 위해 코드블럭으로 작성한다.
- 고객 메시지에는 마크다운 볼드(`**`)를 쓰지 않는다.
- 상품명 강조는 큰따옴표(`"상품명"`)를 사용한다.
- 첫 문장은 반드시 아래 형식을 기본으로 한다.
  - `[마켓명]에서 주문하신 "[상품명]" 관련하여 안내 말씀 드립니다.`
- `주문해 주셔서 감사합니다` 문구는 사용하지 않는다.
- 주문정보에 `010`으로 시작하는 번호가 있으면, 본문 아래 별도 코드블럭으로 번호만 따로 출력한다.
- 취소 안내 시 `직접` 표현은 사용하지 않는다.

## 2. 공통 기본형

```text
안녕하세요, 고객님.

[마켓명]에서 주문하신 "[상품명]" 관련하여 안내 말씀 드립니다.

[상황 설명 및 원인]

[해결 방안 또는 대안]

[추가 안내 / 후속 조치]

감사합니다.
```

## 3. 품절 / 재고 부족 안내

### 공통 원칙
- 먼저 사과
- 대체 옵션이 있으면 먼저 제시
- 취소는 고객 선택으로 유도
- 빠른 환불을 원하면 마켓 주문내역 취소 경로 안내
- `직접 취소가 어려우신 경우` 표현 금지

### 스마트스토어 / 네이버 공통 문구 구조

```text
안녕하세요, 고객님.

스마트스토어에서 주문하신 "[상품명]" 관련하여 안내 말씀 드립니다.

주문하신 상품의 [옵션명] 재고가 현재 확보되지 않아 출고가 어려운 것으로 확인되었습니다.
불편을 드려 죄송합니다.

[대체 옵션 제안]

변경을 원하지 않으시는 경우, "취소요청"이라고 회신 부탁드립니다.
(확인 후 순차적으로 취소 처리되며, 환불까지 영업일 기준 5~7일 소요될 수 있습니다.)

즉시 환불을 원하시면 아래 링크를 통해 취소 진행 부탁드립니다.
▶ https://pay.naver.com/history?serviceChannel=SHOPPING
▶ 네이버 앱: 주문목록 → 취소 및 환불 요청

감사합니다.
```

### 쿠팡 공통 문구 구조

```text
안녕하세요, 고객님.

쿠팡에서 주문하신 "[상품명]" 관련하여 안내 말씀 드립니다.

불편을 드려 죄송합니다.

[대체 옵션 제안]

변경을 원하지 않으시는 경우, "취소요청"이라고 회신 부탁드립니다.
(확인 후 순차적으로 취소 처리되며, 환불까지 영업일 기준 5~7일 소요될 수 있습니다.)

즉시 환불을 원하실 경우 아래 링크 또는 쿠팡 앱에서 취소 진행이 가능합니다.
▶ https://mc.coupang.com/ssr/mobile/order/list
```

## 4. 통관 정보 오류 안내

### 공통 원칙
- 오류 유형을 명확히 구분
- 유니패스 수정 경로 안내
- 장기 보류 시 비용 발생 가능성 안내

### 개인통관고유부호 오류

```text
안녕하세요, 고객님.

[마켓명]에서 주문하신 "[상품명]" 관련하여 안내 말씀 드립니다.

통관 진행 중 확인 결과, 등록된 개인통관고유부호 정보에 오류 또는 비활성 상태가 확인되었습니다.
번거로우시겠지만 유니패스(unipass.customs.go.kr)에서 개인통관고유부호를 재발급 또는 활성 여부 확인 후 회신 부탁드립니다.

통관 보류가 장기화될 경우 보관료 또는 반송 비용이 발생할 수 있어 빠른 확인 부탁드립니다.

감사합니다.
```

### 우편번호 불일치

```text
안녕하세요, 고객님.

[마켓명]에서 주문하신 "[상품명]" 관련하여 안내 말씀 드립니다.

통관 진행 중 주문서상 주소와 개인통관고유부호에 등록된 주소 정보의 우편번호가 일치하지 않아 통관이 보류되었습니다.

유니패스(unipass.customs.go.kr)에서 등록 주소를 수정하시거나,
개인통관고유부호에 등록된 주소와 동일한 주소로 배송지 변경을 원하시는 경우 회신 부탁드립니다.

통관 보류가 길어질 경우 추가 비용이 발생할 수 있어 빠른 확인 부탁드립니다.

감사합니다.
```

### 전화번호 불일치

```text
안녕하세요, 고객님.

[마켓명]에서 주문하신 "[상품명]" 관련하여 안내 말씀 드립니다.

통관 진행 중 주문서상 연락처와 개인통관고유부호에 등록된 연락처 정보가 일치하지 않아 통관이 보류되었습니다.
번거로우시겠지만 유니패스(unipass.customs.go.kr)에서 등록된 연락처를 확인 및 수정 부탁드립니다.

보류가 장기화될 경우 추가 비용이 발생할 수 있어 빠른 확인 부탁드립니다.
```

## 5. 배송 지연 안내

### 공통 원칙
- 해외창고 / 통관 / 국내배송 단계를 구분해서 설명
- 확인되지 않은 배송 상태 단정 금지
- 필요 시 tracking.tipoasis.com 링크 사용

```text
안녕하세요, 고객님.

[마켓명]에서 주문하신 "[상품명]" 관련하여 안내 말씀 드립니다.

현재 주문 건은 [해외 출고 / 통관 진행 / 국내 배송] 단계로 확인되고 있습니다.
해외 구매대행 상품 특성상 일반적으로 배송은 영업일 기준 7~15일 정도 소요될 수 있으며,
통관 또는 현지 물류 상황에 따라 일정이 다소 변동될 수 있는 점 양해 부탁드립니다.

확인 가능한 배송조회 링크는 아래와 같습니다.
▶ http://tracking.tipoasis.com/[운송장번호]

추가 확인이 필요한 경우 다시 안내드리겠습니다.
감사합니다.
```

## 6. 반품 / 교환 안내

### 단순 변심

```text
안녕하세요, 고객님.

[마켓명]에서 주문하신 "[상품명]" 관련하여 안내 말씀 드립니다.

해당 상품은 해외 구매대행 상품으로, 단순 변심에 의한 반품의 경우 해외 왕복 배송비가 발생할 수 있으며 비용이 상품 금액과 비슷하거나 더 클 수 있습니다.
반품 진행을 원하시는 경우 예상 비용 및 절차 확인 후 안내드리겠습니다.

감사합니다.
```

### 불량 / 파손

```text
안녕하세요, 고객님.

[마켓명]에서 주문하신 "[상품명]" 관련하여 불편을 드려 죄송합니다.

불량/파손 여부 확인을 위해 제품 상태가 잘 보이는 사진 또는 동영상을 보내주시면 확인 후 교환 또는 환불 가능 여부를 빠르게 안내드리겠습니다.

감사합니다.
```

### 고객 귀책 사이즈 교환

```text
안녕하세요, 고객님.

[마켓명]에서 주문하신 "[상품명]" 관련하여 안내 말씀 드립니다.

해외 구매대행 상품 특성상 고객님의 단순 사이즈 선택 오류로 인한 교환은 해외 왕복 배송비 부담이 매우 커 진행이 어려운 점 양해 부탁드립니다.
상품 상세페이지의 사이즈 정보를 다시 확인해주시고, 필요 시 재구매 여부를 검토 부탁드립니다.

감사합니다.
```

## 7. 상품 Q&A 답변 원칙
- 구매 전 질문일 가능성이 높으므로 과도한 확정 표현을 피한다.
- 사이즈/옵션/재고는 상세페이지 또는 실제 재고 확인 기준으로 안내한다.
- 확인 전 추정 답변 금지

### 사이즈 문의

```text
안녕하세요, 또몽이네 스토어입니다.

사이즈는 상품 상세페이지에 안내된 실측 및 표기 기준으로 확인 부탁드립니다.
상품마다 기준이 다를 수 있어 상세페이지 기준으로 확인 후 선택해주시는 것을 권장드립니다.

감사합니다.
```

### 수량/분리 배송 문의

```text
안녕하세요, 또몽이네 스토어입니다.

문의주신 수량 주문 가능 여부 및 분리 배송 가능 여부는 재고 및 출고 조건 확인 후 안내드리겠습니다.
확인 후 다시 답변드리겠습니다.

감사합니다.
```

## 8. 톡톡 문의 답변 원칙
- 대화형 응대이므로 너무 딱딱하지 않게 짧게 답한다.
- 필요한 정보가 부족하면 추가 질문을 먼저 한다.
- 사용불편/불량성 문의는 증상과 환경을 먼저 확인한다.
- 배송문의는 주문/송장 확인 후 조건 검색 결과의 `현재상태 / 통관예상 / 배송예상 / 조회링크`를 우선 반영한다.
- 조건 검색 결과를 확보한 뒤 고객 안내용으로만 자연스럽게 정리하고, 내부 표현은 그대로 노출하지 않는다.

### 사용불편 확인형

```text
안녕하세요 🙂
불편을 드려 죄송합니다.
어떤 기기에서 사용 중이신지와 연결 방식(유선/무선/블루투스 등) 확인 가능하실까요?
확인해주시면 안내 도와드리겠습니다.
```

### 배송기간 문의형

```text
안녕하세요 🙂
해당 상품은 해외직구 상품으로 일반적으로 영업일 기준 7~15일 정도 소요됩니다.
다만 주문 시점 재고나 현지/통관 상황에 따라 조금 달라질 수 있는 점 참고 부탁드립니다.
```

### 배송조회 결과 안내형

```text
안녕하세요, 고객님 😊

[마켓명]에서 주문하신 "[상품명]" 관련하여 배송 현황 안내드립니다.

현재 상태는 [현재상태]로 확인되며,
통관 예상일은 [통관예상], 배송 예상일은 [배송예상]입니다.

아래 링크에서도 확인 가능합니다.
▶ http://tracking.tipoasis.com/[운송장번호]

통관 및 국내 택배사 처리 상황에 따라 일정은 다소 변동될 수 있는 점 참고 부탁드립니다.
감사합니다.
```
