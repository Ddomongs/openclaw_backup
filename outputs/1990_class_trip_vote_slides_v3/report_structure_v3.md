## 2. 후보지 전체 요약

### 1순위 후보: 경주
- 부산 기준: 약 85km / 약 1시간 10분 라우팅, 실제 체감 1시간 20분~1시간 40분 예상
- 성격: 이동·예산·숙소·술자리 밸런스형
- 주요 액티비티: 경주루지월드, 보문단지, 황리단길
- 예상 액티비티 비용: 1인 약 2만~4만원대 확인 필요
- 장점: 15만원 예산 안에서 가장 안정적, 비 와도 대체 일정 가능
- 단점: 새로움은 약할 수 있음

### 2순위 후보: 거제·통영
- 부산 기준: 통영 약 91km / 약 1시간 26분 라우팅, 실제 체감 1시간 30분~2시간 예상
- 성격: 바다 + 카트/루지 + 단체 숙소 현실픽
- 주요 액티비티: 더카트인 통영, 통영 루지, 요트/낚시/해안 드라이브
- 확인 가격: 더카트인 통영 Adult 서킷 1회 25,000원 / 2회 45,000원 / 3회 70,000원
- 장점: 부산팀 이동 부담 낮음, 활동성과 바다 여행 느낌을 동시에 확보
- 단점: 날씨 영향, 성수기 숙소비 상승 가능

### 3순위 후보: 여수
- 부산 기준: 약 177km / 약 2시간 31분 라우팅, 실제 체감 2시간 50분~3시간 30분 예상
- 성격: 낭만, 맛집, 밤 분위기 만족도 후보
- 주요 액티비티: 해상 케이블카, 유월드 루지/테마파크, 레일바이크, 낭만포차
- 예상 액티비티 비용: 케이블카 1만~2만원대, 유월드 패스/루지 계열 3만~6만원대 확인 필요
- 장점: 여행 온 느낌 강함, 밤 일정 만족도 높음
- 단점: 15만원 예산에서는 숙소·식비 조절 필요

### 4순위 후보: 단양·제천
- 부산 기준: 단양 약 271km / 약 3시간 31분 라우팅, 실제 체감 4시간 전후 예상
- 성격: 액티비티 강한 도전형 후보
- 주요 액티비티: 패러글라이딩, 카트, 수상레저, 카페산, 도담삼봉
- 예상 액티비티 비용: 패러글라이딩 1인 약 9만~13만원대 확인 필요
- 장점: 사진/영상, 기억에 남는 여행 가능
- 단점: 이동시간 길고, 패러글라이딩 포함 시 15만원 초과 가능성 큼

### 참고 후보: 강릉·양양
- 부산 기준: 강릉 약 348km / 약 5시간 6분, 양양 약 395km / 약 5시간 37분 라우팅. 실제 체감 5시간 30분~7시간 예상
- 성격: 서핑·동해 감성 특별픽
- 주요 액티비티: 서핑 강습, 바다 카페, 펍, 해변 일정
- 예상 액티비티 비용: 서핑 강습+렌탈 1인 약 5만~8만원대 확인 필요
- 장점: 가장 특별하고 여행 분위기 강함
- 단점: 1박2일 기준 왕복 이동 부담이 너무 큼, 15만원 초과 가능성 높음

## 3. 부산 기준 이동거리·시간 보고 방식

### 발표 핵심 문장
“이번 여행은 부산 출발 인원이 많기 때문에, 왕복 이동시간이 실제 만족도를 크게 좌우합니다. 1박2일 기준으로는 편도 2시간 이내 후보가 가장 안정적입니다.”

### 비교 기준
- 경주: 약 85km / 편도 약 1시간 10분 라우팅 / 현실 체감 1시간 20분~1시간 40분
- 거제·통영: 약 91km / 편도 약 1시간 26분 라우팅 / 현실 체감 1시간 30분~2시간
- 여수: 약 177km / 편도 약 2시간 31분 라우팅 / 현실 체감 2시간 50분~3시간 30분
- 단양·제천: 약 271~296km / 편도 약 3시간 31분~3시간 44분 라우팅 / 현실 체감 4시간 전후
- 강릉·양양: 약 348~395km / 편도 약 5시간 6분~5시간 37분 라우팅 / 현실 체감 5시간 30분~7시간

### 결론
- 이동 안정권: 경주, 거제·통영
- 이동 감수 가능권: 여수
- 이동 부담권: 단양·제천
- 1박2일 비효율권: 강릉·양양

## 4. 15만원 예산 기준

### 1인 예산 배분안
- 숙소: 5만~7만원
- 식사/술/장보기: 5만~6만원
- 액티비티: 2만~4만원
- 교통/기타: 1만~2만원

### 예산 판단
- 경주: 가장 안정적
- 거제·통영: 카트 1회 기준이면 안정적, 2회 이상 또는 고급 숙소면 조정 필요
- 여수: 가능하지만 숙소·식비 조절 필요
- 단양·제천: 패러글라이딩 포함 시 예산 초과 가능성 큼
- 강릉·양양: 서핑+장거리 이동으로 초과 가능성 큼
