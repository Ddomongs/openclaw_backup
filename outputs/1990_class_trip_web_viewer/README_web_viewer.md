# 웹 뷰어 사용 안내

생성 위치:
/Users/dh/.openclaw/workspace/outputs/1990_class_trip_web_viewer/index.html

구성:
- index.html: 모바일/PC 웹 뷰어
- images/: 1~20번 슬라이드 이미지
- slides.json: 슬라이드 목록

사용 방법:
1. Mac/PC에서 index.html을 더블클릭하면 브라우저로 볼 수 있습니다.
2. 카카오톡에 URL로 공유하려면 이 폴더 전체를 웹호스팅에 올려야 합니다.
3. 가능한 배포 방식:
   - Netlify Drop: 폴더 업로드 후 즉시 공유 URL 생성
   - GitHub Pages: 저장소 공개/Pages 설정 필요
   - 개인 서버: 폴더를 웹 루트에 업로드

주의:
- 카카오톡에 HTML 파일 자체를 보내는 것보다 URL 공유가 훨씬 보기 좋습니다.
- 외부 공개 업로드는 대표님 승인 후 진행하는 것을 권장합니다.
