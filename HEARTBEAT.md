# HEARTBEAT.md — 정기 작업

## 매 하트비트마다 확인
- 이 하트비트의 스마트스토어/문의 점검은 모두 `CS/스마트스토어 작업`으로 간주한다.
- 시작 전 반드시 `./scripts/browser-mcp/browser-ensure-ready-cs.sh`를 먼저 실행해 `ready`를 확인하고, 자동화 Chrome(9223)만 사용한다.
- 9222 일반 Chrome(user 프로필) attach/재시작 복구는 하트비트 점검에서 사용하지 않는다.
- 자동 하트비트에서는 browser 도구로 기존 Chrome(user 프로필)을 건드리지 않는다.
- 9223 전용 로컬 스크립트가 없는 자동 점검 항목은 무리하게 브라우저 조작하지 말고 건너뛴다.
- 자동 하트비트는 스마트스토어 로그인 버튼/간편 로그인 버튼을 클릭하지 않는다. 로그인 필요 상태면 알림만 남기고 중단한다.
- 스마트스토어 대시보드의 미답변 문의 위젯 기준으로 잔여 건수를 확인한다.
- 대상은 `톡톡 문의`, `주문 고객 문의`, `상품 Q&A`다.
- 직전 보고 대비 건수가 증가한 항목이 있을 때만 Discord 대화방에 알린다.
- 동일 건수 반복 상태는 다시 알리지 않는다.
- 브라우저/게이트웨이 제어가 불가능하면 상태 변경 시 1회만 알리고, 동일 실패 상태는 반복 보고하지 않는다.
